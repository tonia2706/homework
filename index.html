Веб-сайти - це основа, на якій тримається інтернет, і всі вони побудовані на трьох основних технологіях. HTML - відповідає за наповнення веб-сторінки контентом, структуру і зміст. CSS - використовується для оформлення та позиціонування, зовнішній вигляд. JavaScript - дозволяє додати функціонал, реакцію на дії користувача.
